# First pass at an ECS engine

See `example/demo.js` for current usage.

## Installation

Just use npm to install dependencies.

```bash
$ npm install
```

## Running the demo

### Build

```bash
$ npm run build-demo
```

Or, to play with the demo code:

```bash
$ npm run watch-demo
```

### Running

Just open `example/index.html` in a browser.
